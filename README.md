# 🎓 Student Performance Analysis Dashboard (Power BI)

## 📌 Project Overview

This project presents an interactive **Power BI dashboard** designed to analyze student academic performance. The analysis focuses on **Math, Reading, and Writing scores** while examining how different factors such as gender, parental education level, lunch type, and test preparation courses influence student outcomes.

The goal of this project is to transform raw data into **meaningful insights using data visualization techniques**.

## 📊 Dataset Information

The dataset used in this project contains information about student academic performance. It includes the following fields:

* Gender
* Race/Ethnicity
* Parental Level of Education
* Lunch Type
* Test Preparation Course
* Math Score
* Reading Score
* Writing Score

This dataset allows us to explore different factors that may influence student academic outcomes.

## 🛠 Tools & Technologies

The following tools were used to complete this project:

* **Power BI** – Data visualization and dashboard creation
* **Excel / CSV Dataset** – Data source
* **Basic DAX Measures** – Used to calculate KPIs and averages
* **GitHub** – Project documentation and version control

## 📈 Dashboard Features

The Power BI dashboard includes several interactive visualizations:

* **KPI Cards** showing average scores across subjects
* **Bar Charts** comparing performance by gender
* **Column Charts** analyzing parental education influence
* **Pie Charts** showing lunch distribution
* **Scatter Plot** exploring relationships between reading and writing scores
* **Tables and matrices** summarizing student performance

These visuals help transform raw data into easy-to-understand insights.

---

## 📊 Business Questions

This dashboard was designed to answer several analytical questions about student academic performance:

1. What is the overall average performance across Math, Reading, and Writing?
2. Do male and female students perform differently across subjects?
3. How does parental education level influence student academic performance?
4. Does completing a test preparation course improve student scores?
5. What relationship exists between reading and writing scores?
6. Does lunch type (standard vs free/reduced) influence student performance?
7. Which subject shows the highest average score among students?
8. What patterns can be observed across different race/ethnicity groups?

These questions help guide the analysis and highlight key patterns within the data.

---

## 🔍 Key Insights

Some important insights from the analysis include:

* Students who **completed the test preparation course** tend to perform better on average.
* There is a **strong correlation between reading and writing scores**.
* **Parental education level** appears to influence student academic performance.
* Students with **standard lunch** generally perform better than those with free or reduced lunch.
